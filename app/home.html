Trigger from SuccessFactors

Use this webhook in SAP SuccessFactors → Integration Center to trigger your DocuSign Maestro workflow. It uses your DocuSign access token.

  1. In Integration Center, create a More Object Types → REST (outbound) integration.
  2. Set the destination REST API URL to the Webhook URL above and method to POST.
  3. Set Content-Type to application/json (no authentication required).
  4. Use the Request Body above, mapping SuccessFactors fields as additional properties if needed.
  5. Save & run — the Maestro workflow will be triggered for your account.
